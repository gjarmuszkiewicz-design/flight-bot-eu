# Flight Finder EU Bot

Telegram bot do wyszukiwania tanich lotów w Europie (UE).

## Funkcje
- Wyszukiwanie lotów bezpośrednich
- Loty z przesiadkami
- Porównanie cen
- AI-powered search z Claude

## Tech Stack
- Python 3.11
- python-telegram-bot
- Amadeus Flight API
- Claude AI (Anthropic)

## Deploy
Railway.app (automatic deployment)
